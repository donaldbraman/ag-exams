**Q6.** Instead of depraved heart murder, prosecutors charge Arthur with felony murder, using drug distribution as the predicate felony. How does California's felony murder approach apply to Arthur's drug distribution predicate?

(a) The felony survives the merger doctrine because distribution has an independent purpose, but it may fail the inherently dangerous test if the statute can theoretically be violated safely. <!-- correct -->
(b) The felony merges with the homicide because drug distribution is an assaultive act, but it satisfies the inherently dangerous test based on the specific lethal facts.
(c) The felony survives the inherently dangerous test because all drug offenses presume lethality, but it merges because the distribution was the direct cause of the death.
(d) The felony fails both tests because distributing narcotics lacks an independent felonious purpose and, looking at the facts as committed, this specific transaction was not violently coercive.
(e) The felony survives both tests because drug distribution is malum in se, which exempts it from the merger doctrine and automatically satisfies the inherently dangerous requirement.

**Answer:** (a)

**Explanation:** (a) is correct because California uses a two-part limitation: the felony must have an independent purpose to survive the merger doctrine (drug distribution does, as its purpose is financial, not assaultive), and it must be inherently dangerous in the abstract (which distribution may fail if it can theoretically be committed safely). (b) fails because California uses an abstract elements test for inherent dangerousness, not a facts-as-committed test. (c) fails because while it correctly identifies the abstract elements test, it ignores that the prompt requires analyzing both limitations, including merger. (d) fails because distributing narcotics is not considered an assaultive act that merges with the homicide; its primary purpose is not violent injury. (e) fails because malum in se classification does not exempt a felony from the merger doctrine or the inherently dangerous test.
